# BreadCrumb-SWE3090XA-Project
BreadCrumb is a mobile price comparison app for consumers. It lets users compare prices across supermarkets, pharmacies, electronics, furniture, and liquor stores.

/*BreadCrumb*/is named after the idea of following a trail to the best deals. The platform helps shoppers track prices, promotions, and products across multiple stores to make smarter purchasing decisions.
