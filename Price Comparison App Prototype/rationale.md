# Project Rationale

## Introduction

BreadCrumb was developed to address the challenge consumers face when trying to identify the best prices for products sold across multiple retail stores.

Many shoppers compare prices manually by visiting different stores, browsing catalogues, or checking multiple websites. This process can be time-consuming and inefficient.

The BreadCrumb application aims to simplify this process by providing a single platform where users can search for products, compare prices, and discover opportunities to save money.

---

## Why the Project Was Chosen

Price differences between stores can significantly affect household spending.

Consumers often purchase products without knowing that the same item may be available at a lower price elsewhere.

BreadCrumb was chosen as a project because it offers a practical solution to a real-world problem while demonstrating software engineering concepts such as:

- User interface design
- User experience design
- Information architecture
- Data organization
- System analysis and design

---

## Target Users

The application is intended for:

- Students
- Families
- Individual shoppers
- Budget-conscious consumers
- Retail customers in Eswatini

---

## Design Objectives

The design was guided by the following objectives:

### Simplicity

Users should be able to compare prices quickly with minimal navigation.

### Accessibility

Large buttons, clear labels, and simple layouts improve usability for all users.

### Efficiency

The application minimizes the time required to locate products and compare prices.

### Consistency

Similar colors, icons, and layouts are used throughout the application to create a consistent user experience.

### Savings Awareness

The system highlights cheaper alternatives, promotions, and combo deals to encourage informed purchasing decisions.

---

## Design Decisions

### Home Dashboard

The dashboard provides quick access to frequently used functions such as search, price comparison, shopping lists, and alerts.

### Search Page

The search page allows users to locate products easily and view available prices across multiple stores.

### Comparison Page

This page displays product prices side by side, enabling users to identify the most affordable option.

### Shopping Lists

Users can organize products before shopping and estimate costs in advance.

### Combo Deals

Combo deals were included to showcase bundled promotions and increase potential savings.

### Price Alerts

Alerts notify users when tracked products decrease in price or become available on promotion.

### Profile Page

The profile page centralizes user information and application preferences.

---

## Expected Impact

The application is expected to:

- Improve shopping efficiency
- Increase price transparency
- Support better purchasing decisions
- Help users reduce shopping expenses
- Encourage smarter spending habits

---

## Conclusion

BreadCrumb demonstrates how technology can be used to simplify everyday shopping activities. By bringing price comparison, shopping lists, combo deals, and alerts into a single platform, the application provides users with a convenient and effective way to make informed purchasing decisions and maximize savings.
